# Django Project - flip

This is a clean, ready-to-run Django project including:
- accounts app
- base app
- products app

## Run instructions

```bash
cd flip_project_clean
python -m venv venv
venv\Scripts\activate    # On Windows
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

Open http://127.0.0.1:8000/ in your browser.
